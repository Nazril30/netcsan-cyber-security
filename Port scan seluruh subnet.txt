1. Perintah Utama (Kombinasi Auto Subnet + Ping Sweep + Port Scan)
Mode Otomatis (Rekomendasi):
python netscan.py

Mode Manual (Jika ingin menguji subnet spesifik):
python netscan.py 192.168.1.0/24

2. Perintah Pemetaan MAC Address (LAN Scan):
python netscan.py --lanscan

3. Perintah Sistem Deteksi Intrusi (IDS):
python netscan.py --ids
